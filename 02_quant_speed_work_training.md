# Module 2: Work, Speed & Measurement

## 1. Time & Work
**Q1. A in 15 days, B in 20 days. Work together 4 days. Fraction left?**
**Sol:**
$LCM(15,20) = 60U$ (Total Work)
$A = 4U/d$
$B = 3U/d$
Together = $7U/d$
Work done in 4d = $7 \times 4 = 28U$
Left = $60 - 28 = 32U$
Fraction left = $32 / 60 = 8/15$

**Q2. A is twice as good as B. Together = 18 days. A alone = ?**
**Sol:**
Eff ratio A:B = $2:1$
Total Eff = $3$
Total Work = $18 \times 3 = 54U$
Time for A = $54 / 2 = 27$ days

**Q3. 39 men, 12 days, 5 hrs/day. 30 men, 6 hrs/day -> ? days**
**Sol:**
$M_1 D_1 H_1 = M_2 D_2 H_2$
$39 \times 12 \times 5 = 30 \times D_2 \times 6$
$2340 = 180 \times D_2$
$D_2 = 13$ days

**Q4. (A+B)=30, (B+C)=24, (C+A)=20. Work together 10 days, B & C leave. A finish remaining in?**
**Sol:**
$LCM(30, 24, 20) = 120U$
$(A+B) = 4$
$(B+C) = 5$
$(C+A) = 6$
$2(A+B+C) = 15$
$A+B+C = 7.5$
$A = 7.5 - 5 = 2.5$
Work in 10d = $10 \times 7.5 = 75U$
Left = $120 - 75 = 45U$
Time for A = $45 / 2.5 = 18$ days

**Q5. P=8hr, Q=10hr, R=12hr. Start 9am. P close 11am. Finished at?**
**Sol:**
$LCM(8,10,12) = 120U$
$P=15$, $Q=12$, $R=10$
Combined = $37$
Work (9am-11am) = $37 \times 2 = 74U$
Left = $120 - 74 = 46U$
$Q+R = 22U/hr$
Time = $46 / 22 \approx 2$ hrs
$11am + 2hrs = 1pm$

## 2. Pipes & Cisterns
**Q1. A=20min, B=30min. Together = ?**
**Sol:**
$LCM(20, 30) = 60U$
$A = 3$
$B = 2$
$Total = 5$
Time = $60 / 5 = 12$ min

**Q2. Pump fills in 2hr. With leak = 7/3 hr. Leak empties in?**
**Sol:**
Let Work = $14U$
Pump = $14/2 = 7U/hr$
Net = $14 / (7/3) = 6U/hr$
Leak = $Net - Pump = 6 - 7 = -1U/hr$
Time = $14 / 1 = 14$ hrs

**Q3. A=37.5m (75/2), B=45m. Cistern filled in 30m. B turned off after?**
**Sol:**
$LCM(75, 45) = 225U$
$A = 225 / (75/2) = 6$
$B = 225 / 45 = 5$
A runs full 30m $\rightarrow 30 \times 6 = 180U$
Left = $225 - 180 = 45U$
B runs = $45 / 5 = 9$ min

**Q4. A=30, B=20, C=10 min. Discharge P,Q,R. Proportion of R after 3 mins?**
**Sol:**
Efficiencies (Flow rates) -> $A=1/30, B=1/20, C=1/10$
Ratio of flow $P:Q:R = 2:3:6$
Total flow parts = $11$
Proportion of R = $6/11$

**Q5. Tank 2/5 full. A fills in 10m (+3U), B empties in 6m (-5U). Net?**
**Sol:**
Capacity = $30U$
Present = $2/5 \times 30 = 12U$
Net Eff = $3 - 5 = -2U/m$
Empty time = $12 / 2 = 6$ min

## 3. Time, Speed & Distance
**Q1. 240 kmph in 5 hrs. Same distance in 5/3 hrs speed = ?**
**Sol:**
$Dist = 240 \times 5 = 1200$ km
$New\ Time = 5/3$ hrs
$Speed = 1200 / (5/3) = 720$ kmph

**Q2. 2/15 rail, 9/20 bus, 10km cycle. Total = ?**
**Sol:**
Fraction covered = $2/15 + 9/20 = 8/60 + 27/60 = 35/60 = 7/12$
Remaining = $1 - 7/12 = 5/12$
$5/12(D) = 10$
$D = 24$ km

**Q3. Speed 54 kmph, with stoppages 45 kmph. Stoppage mins/hr?**
**Sol:**
$Stoppage = \frac{Diff}{Speed} \times 60$
$= \frac{54-45}{54} \times 60$
$= \frac{9}{54} \times 60$
$= 10$ mins

**Q4. Walk 14 km/hr instead of 10 km/hr -> 20km more. Actual dist?**
**Sol:**
$\frac{x}{10} = \frac{x+20}{14}$
$14x = 10x + 200$
$4x = 200$
$x = 50$ km

**Q5. Speed=5/7 of actual covers 42km in 1h 40m 48s. Actual speed?**
**Sol:**
$Time = 1 + 40/60 + 48/3600$
$= 1 + 2/3 + 1/75$
$= 126/75$ hrs
$New Speed = 42 / (126/75) = 25$ kmph
$Actual \times (5/7) = 25$
$Actual = 35$ kmph

## 4. Trains, Boats & Streams
**Q1. Train1=27s, Train2=17s cross man. Cross each other=23s. Speed ratio?**
**Sol:**
Alligation on crossing times:
$T_1 = 27$
$T_2 = 17$
$Middle = 23$
Ratio = $(23-17) : (27-23)$
$= 6 : 4$
$= 3 : 2$

**Q2. Train 125m passes man running 5kmph same dir in 10s. Train speed?**
**Sol:**
$Rel\ Speed = (S - 5) \times 5/18$
$Dist = 125$
$Time = 10$
$(S - 5) \times 5/18 = 125 / 10 = 12.5$
$S - 5 = 12.5 \times 18 / 5$
$S - 5 = 45$
$S = 50$ kmph

**Q3. Boat=13, Stream=4. Time for 68km downstream?**
**Sol:**
$Down\ Speed = 13 + 4 = 17$ kmph
$Time = 68 / 17 = 4$ hrs

**Q4. Downstream=15, Current=2.5. Upstream=?**
**Sol:**
$u + v = 15$
$u + 2.5 = 15$
$u = 12.5$
$Up = u - v$
$Up = 12.5 - 2.5 = 10$ kmph

**Q5. Boat=15. 30km down + 30km up = 4.5 hrs. Stream=?**
**Sol:**
$\frac{30}{15+v} + \frac{30}{15-v} = 4.5$
Test options, e.g., $v=5$:
$\frac{30}{20} + \frac{30}{10} = 1.5 + 3 = 4.5$ (Matches)
Stream = $5$ kmph
